## Installation Guide

### Windows
Launch the `.exe` installer. Windows SmartScreen may warn you because the app is currently unsignedâ€”click "More info" then "Run anyway".

### macOS
1. Download the `.zip` for your architecture.
2. Extract and move `KafkaLens.app` to your `/Applications` folder.
3. If macOS blocks the app, go to **System Settings > Privacy & Security** and click **Open Anyway**.

### Linux
1. Extract the `.zip` archive.
2. Ensure the `KafkaLens` binary has execution permissions (`chmod +x KafkaLens`).
3. Run the binary directly.
