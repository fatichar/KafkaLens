## Installation Instructions

### Windows
Run the `.exe` installer and follow the prompts.

### Linux
1. Download the `.zip` file.
2. Extract it to your preferred location.
3. Run the `KafkaLens` executable.

### MacOS
1. Download the `.zip` file.
2. Extract it.
3. Move `KafkaLens.app` to your `Applications` folder.
4. (Note) You may need to allow the app to run in "Security & Privacy" settings as it is currently not signed.
