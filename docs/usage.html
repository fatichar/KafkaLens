Usage Documentation

Master KafkaLens with our comprehensive guide to managing clusters and exploring data.

🏗️ Getting Started

1. Managing Clusters

To begin, you need to configure your Kafka clusters. KafkaLens supports both direct and remote connections.

  1. Click Cluster -> Edit Clusters (or press Ctrl+E).
  2. Local Clusters: Add a cluster by providing a name and bootstrap servers (e.g., localhost:9092).
  3. Remote Gateways: Add a gRPC client to connect to clusters behind a firewall or in a VPC.

[Screenshot: Cluster Management Dialog showing Add/Edit options]

2. Navigation

The sidebar displays a hierarchical tree of your configured clusters. Click a cluster to connect and expand it to see:

  • Topics: Use the search bar at the top of the tree to filter topics by name.
  • Partitions: Drill down into specific partitions to inspect message distribution and lag.

[Screenshot: Sidebar showing Cluster Tree with Topic Search]

🔍 Exploring Data

Fetching Messages

Click a topic or partition to open its data view. Use the 'Fetch Panel' on the right to load data:

  • Load Latest: Fetches the most recent messages from the topic.
  • Seek by Offset: Jump to a specific offset for precise debugging.
  • Seek by Time: Jump to messages from a specific point in time (e.g., "5 minutes ago" or a specific date).

Advanced Filtering

Filter through thousands of messages instantly using the search bar. We support powerful boolean logic:

  • Simple Match: error
  • Logical AND/OR: error && !timeout or "order created" || "payment success"
  • Grouping: (error || warning) && user_id_123

[Screenshot: Message View with Filter Bar active]

🔬 Deep Dive into Messages

Message Inspection

Select any message to view its full details in the inspection panel. KafkaLens automatically detects and formats data:

  • JSON: Pretty-printed, collapsible, and searchable.
  • Text: Raw UTF-8 string display.
  • Numeric: Interpret bytes as Int32, Int64, Float, or Double.

[Screenshot: Message Detail Panel with JSON Formatter]

Headers & Metadata

Inspect Kafka headers and metadata (Partition, Offset, Key, Timestamp) alongside the message payload. Right-click any value to copy it to your clipboard.

⚡ Power Features

Plugin Manager

Extend KafkaLens with custom decoders. If you use proprietary formats (like custom Avro or Protobuf), you can install or develop plugins to visualize them as JSON.

[Screenshot: Plugin Manager showing installed decoders]

Saving & Loading

Export messages for offline analysis or sharing:

  • Save Message: Right-click a message and choose Save Raw (.klm) or Save Formatted (.txt).
  • Open Saved: Use Ctrl+O to open and inspect .klm files as if they were live data.

💡 Shortcuts & Tips

Ctrl + E Edit Clusters
Ctrl + O Open Saved Message
Ctrl + W Close Current Tab
Ctrl + F Focus Search Bar
F5 Refresh/Load Latest
